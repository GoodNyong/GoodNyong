# GoodNyong

안녕하세요. 서비스의 구조를 이해하고, 필요한 기능을 끝까지 구현하는 백엔드 개발자입니다.

**Campfire, Blinkos, Mealhub, one-for-logis, high-tension** 프로젝트를 진행하며  
커뮤니티, 헬스케어, 배달, 물류, 커머스 도메인을 경험했습니다.  
사용자 기능 개발부터 인증/인가, 데이터 처리, API 연동, 비동기 처리, 운영 환경 구성까지 폭넓게 다뤘습니다.

## Tech Stack
- **Backend**: Java, Spring Boot, Spring Security, JPA, QueryDSL
- **Database**: MySQL, PostgreSQL, Redis, MongoDB
- **Infra / DevOps**: Docker, GitHub Actions
- **Messaging / Monitoring**: Kafka, ELK Stack
- **Frontend**: JavaScript, JSP, JSTL, HTML/CSS

## Projects
### Campfire
게임 커뮤니티 웹 애플리케이션.  
JSP/Servlet 기반으로 회원 관리, 게시판/댓글, 리뷰, 외부 API 연동 기능을 구현했습니다.

### Blinkos
건강 관리 웹 애플리케이션.  
Spring MVC 기반으로 회원 관리, 건강 기록, 커뮤니티, 관리자 기능을 구현했습니다.

### Mealhub
음식 주문 플랫폼.  
주문, 결제, 리뷰 등 주요 도메인을 중심으로 백엔드 기능을 구현했습니다.

### one-for-logis
스마트 통합 물류 플랫폼.  
MSA 구조에서 주문, 배송, 알림, 권한 관리, Kafka/Redis 기반 처리 흐름을 경험했습니다.

### high-tension
C2C 온라인 쇼핑 플랫폼.  
이커머스 핵심 흐름과 오케스트레이션, 모니터링, 이벤트 기반 아키텍처를 다뤘습니다.

## Strengths
- 요구사항을 구조적으로 정리하고 기능으로 구현하는 역량
- 단일 서비스부터 MSA까지 폭넓게 경험한 백엔드 개발
- 협업 과정에서 유지보수성과 확장성을 고려하는 개발 방식
